# Openframe

Openframe 是一个面向 AI 漫剧/剧情短视频的创作工作台，支持 Web 与桌面端。

[English](./README.en.md)

## 项目功能

- 一站式流程：项目 -> 剧本 -> 角色/道具/场景 -> 分镜 -> 生产/导出
- 剧本编辑 + AI 辅助：自动补全、想法生成剧本、小说片段改编、润色重写
- 结构化提取：从剧本自动提取角色、道具、场景与分镜信息
- 人物关系图谱：基于剧本提取关系，并支持持续优化
- 分镜生成：可按剧本自动生成，支持指定镜头数量
- 画面与视频生产：支持关键帧/镜头视频生成与时间线编排
- 导出能力：支持合并视频、FCPXML、EDL（便于接入 PR/达芬奇）
- AI 服务配置：支持文本、生图、视频三类模型与服务地址独立配置
- 存储能力：支持本地存储与 S3/COS/OSS 对象存储

## Web 端

- 浏览器直接可用，无需安装客户端
- 使用浏览器本地存储保存项目数据
- 支持 AI 生产与导出流程

## Desktop 端

- 本地桌面应用，数据默认本地存储
- 支持本地目录相关能力与原生导出体验
